CVE-2023-23397 - Microsoft Outlook Elevation of Privilege Vulnerability

Description:
Une vulnérabilité critique dans Microsoft Outlook permet l’élévation de privilèges sans interaction de l’utilisateur. Un attaquant peut envoyer un email piégé qui déclenche automatiquement une connexion NTLM vers un serveur contrôlé à distance.

Score CVSS : 9.8 (Critical)
Composant affecté : Outlook
Exploitation : Sans clic, à la réception de l’email

Recommandations :
- Appliquer les patchs de sécurité fournis par Microsoft (mars 2023)
- Bloquer les ports de communication non autorisés sortants (ex : 445, 139)
- Surveiller les connexions NTLM suspectes dans les journaux

CVE-2022-30190 - Follina - Remote Code Execution via MSDT

Description:
Cette vulnérabilité permet l’exécution de code à distance via l’appel MSDT dans Microsoft Office. Déclenchée par des documents Word malveillants contenant des liens vers des scripts.

Score CVSS : 7.8 (High)
Composant affecté : Office / MSDT
Exploitation : Document piégé

Recommandations :
- Désactiver MSDT via la base de registre
- Mettre à jour Office avec les correctifs de juin 2022
- Bloquer les documents RTF ou DOC externes

CVE-2021-44228 - Log4Shell - RCE dans Apache Log4j

Description:
Injection de chaînes JNDI dans les journaux, menant à une exécution de code à distance sur les serveurs utilisant Log4j < 2.15.0.

Score CVSS : 10.0 (Critical)
Composant affecté : Log4j
Exploitation : Chaînes utilisateur mal filtrées

Recommandations :
- Mettre à jour vers Log4j 2.17.0+
- Désactiver les appels JNDI
- Filtrer toutes les entrées utilisateurs
